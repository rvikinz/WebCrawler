1. Please enter following input while running this script by double clicking on it
    i)   Number of hopes  
    ii)  Total number of pages to be download
    iii) Path of Document store
    iv)  Seed URL path file ( Inlcuding name of text file )
    v)   Number of maximum threads.
 
2.   Examples of input

	i)   Number of hopes: 12 ( any Integer )

	ii)  Total Number of pages to be downloaded: 999 ( any Integer )

	iii) Path of Document Store - 
		Give the absolute path of directory
		e.g.  /Users/vikashkumar/Documents/IR/Download

	iv)  Seed URL path
		Give the absolute path of text file of seed URL
		e.g /Users/vikashkumar/Documents/SeedURL/seedURL.txt

	v)   Number of maximum threads: 100 ( any Integer )

3. If Double clicking not work 
    i)      Please open your terminal
    ii)     Go to directory location of script
    iii)    Run this command - $sh crawler.sh
